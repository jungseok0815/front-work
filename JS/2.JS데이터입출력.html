2. 데이터를 입력받는 구문 (변수에 기록 가능)

1)변수 = [window].confirm("질문내용");

confirm 호출시 "질문내용" 확인/취소 버튼이 존재하는 알림창 발생 확인버튼 클릭시 true 취소버튼 클릭시 false를 반환

2)변수 = [window].prompt("질문내용");

prompt 호출시 "질문내용"과 입력할 수 있는 "텍스트 상자", "확인/취소"이 보여지는 알림창 발생 확인 버튼 클릭시 텍스트 상자에 입력되어있는 값이 문자열로 반환 취소버튼 클릭시 null 반환

4)변수 = 선택한 input요소.value;
